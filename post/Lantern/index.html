Lantern V2.0

发表于 2015-09-16 更新于 2018-01-15 分类于 信息安全 阅读次数：

⚠️Lantern 3.0 beta开始有收费的苗头了，所以我现在已经基本不用Lantern转用SS了😂

我一直会推荐朋友用Lantern进行翻墙，最新的V2.0不需要Java环境，并且安装特别简单。

上面是我在实验室门口拍的Lantern ;)

有一个需要注意的地方：Lantern不能自动更新，如果使用遇到了问题，请重新下载安装包进行更新。

Github：https://github.com/getlantern/lantern

Lantern 的官方安装包

  • Windows XP SP 3 and above
  • OSX 10.8 and above
  • Ubuntu 14.04 32 bit
  • Ubuntu 14.04 64 bit
  • Arch Linux
